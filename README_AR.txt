مخزن الأشموني - نسخة أونلاين حقيقية
===================================

دي النسخة المناسبة للشبكة بين مجموعة أشخاص.

المميزات:
- تسجيل دخول واسم مستخدم وكلمة مرور
- قاعدة بيانات مشتركة
- نفس البيانات تظهر عند كل الأجهزة
- جاهزة للرفع على Render أو أي استضافة Python
- مناسبة لربطها مع تطبيق أندرويد

التشغيل المحلي:
1) python -m venv .venv
2) فعّل البيئة
3) pip install -r requirements.txt
4) python app.py
5) افتح /init-db مرة واحدة
6) سجل دخول admin / 123456

الرفع أونلاين:
- استخدم render.yaml أو Procfile
- اضبط SECRET_KEY
- اضبط DATABASE_URL على PostgreSQL

مهم:
- لو DATABASE_URL مش موجود، التطبيق هيشتغل SQLite محليًا فقط
- لو عاوزه شبكة فعلًا بين الجميع، لازم PostgreSQL على السيرفر
